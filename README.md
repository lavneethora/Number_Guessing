# Number_Guessing

## Installation and Running

1. **Clone the Repository**:
```bash
git clone https://github.com/lavneethora/Number_Guessing.git
```

2. **Navigate to the Project Directory**:
```bash
cd Number_Guessing
```

3. **Run the Game**:
```bash
python number_guess.py
```
